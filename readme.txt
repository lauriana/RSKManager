Instalação do RSKManager: seguir os passos abaixo devido a descontinuidade da versão do adobe air utilizada para o desenvolvimento do jogo.

1o) Instalar o adobe air para Windows: Adobe AIR 33.1.1.533
2o) Descompactar o arquivo zip do jogo
3o) Clique duas vezes no arquivo de instalação do jogo: RSKM.air 
4o) O jogo está pronto para ser testado em seu computador.
